Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

"position: sticky;" and "position: -webkit-sticky;"

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

"top: 120px;" and "left: 40px". Changing the top and left property values moves the box by the number of pixels specified in order to give the left and top the correct amount of spacing based on its original position.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

"top: 90px;" and "right: 10px;". Changing the top and right property values sets where the box sits on the screen. The box does not move when the user scrolls.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

"bottom: 20px;" and "right: 20px". Changing the bottom and right property values places the box based on those margins inside of its parent element.
